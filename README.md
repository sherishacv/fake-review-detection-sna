# Fake Review Detection using Machine Learning and Social Network Analysis

## 📌 Project Overview

This project detects fraudulent online reviews using Machine Learning and Social Network Analysis (SNA). The system analyzes review text using TF-IDF vectorization and identifies suspicious reviewer behavior through reviewer-product interaction networks.

The project provides an interactive dashboard for visualizing model performance, predictions, and reviewer-product relationships.

---

## 🚀 Features

* Fake vs Genuine Review Classification
* Real-Time Review Prediction
* Interactive Analytics Dashboard
* Model Performance Visualization
* Reviewer–Product Network Graph
* Model Comparison Dashboard
* Precision, Recall and False Alarm Metrics
* Downloadable Detection Reports

## ⚙️ Workflow

1. Load Amazon Reviews Dataset
2. Clean and preprocess review text
3. Generate TF-IDF features
4. Build reviewer-product interaction graph
5. Compute degree centrality scores
6. Train Gradient Boosting Classifier
7. Serve predictions using Flask API
8. Visualize results through dashboard
